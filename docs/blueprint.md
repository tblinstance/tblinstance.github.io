# **App Name**: AuthFlow

## Core Features:

- Login: User login functionality using Firebase Authentication.
- Signup: User signup functionality with email/password and Firebase Authentication.
- Authentication State Management: Manage user authentication state across the application.
- Password Reset: Allow users to reset their password via email.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for trustworthiness and stability.
- Background color: Light gray (#F5F5F5) for a clean and modern look.
- Accent color: Orange (#FF9800) for highlighting interactive elements.
- Body and headline font: 'PT Sans', a modern sans-serif for both headlines and body text.
- Use Material Design Icons from Vuetify for a consistent and recognizable look.
- Employ Vuetify's grid system for responsive layout across different screen sizes.
- Subtle transitions and animations for a smooth user experience, utilizing Vuetify's transition components.
