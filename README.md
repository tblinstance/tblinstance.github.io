# Contabo Subhosting BDT Platform

A premium full-stack platform for reselling Contabo VPS infrastructure with native BDT payment support.

## 🚀 Features
- **React Frontend**: Modern, responsive dashboard with dark theme.
- **Django Backend**: Secure API handling orders, payments, and server provisioning.
- **SSLCommerz Integration**: Real-time BDT payment processing via SSLCommerz.
- **Contabo API**: Automated VPS provisioning.
- **Email Notifications**: Instant alerts for payments and server status.

## 📁 Structure
- `/backend`: Django project with DRF and SSLCommerz logic.
- `/frontend`: React + TypeScript frontend built with Vite.
